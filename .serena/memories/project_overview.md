# ORBOH Project Overview

## Project Purpose
ORBOH is a comprehensive hardware integrated management system that serves as AYA (AI-powered Hardware Assistant). It provides end-to-end support for hardware development projects, including:
- Visual system design with drag-and-drop interface
- Component management and compatibility checking
- AI-powered technical consultation
- Real-time project data persistence

## Tech Stack
- **Frontend**: Next.js 15.2.4, React 19, TypeScript
- **Backend**: Next.js API Routes
- **Database**: PostgreSQL with Prisma ORM 6.11.0
- **Authentication**: NextAuth.js 4.24.11
- **UI Framework**: Tailwind CSS 4, Radix UI components
- **Specialized Libraries**: 
  - @xyflow/react for interactive diagrams
  - OpenAI API for AI assistant
  - Socket.io for real-time features
  - Tiptap for rich text editing

## Development Environment
- Node.js 18+ (recommended: 20+)
- PostgreSQL 13+
- Uses Kiro-style spec-driven development workflow
